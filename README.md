# ADDRESSING-POSITIVITY-VIOLATIONS-VIA-INCREMENTAL-PROPENSITY-SCORE-INTERVENTIONS

This repository contains the code and results for my Master's thesis in Statistics "ADDRESSING POSITIVITY VIOLATIONS VIA INCREMENTAL PROPENSITY SCORE INTERVENTIONS AND ALTERED ESTIMANDS".


The simulation studies were conducted on a MacBook Air with an Apple M1 chip and 16 GB memory using R version 4.5.2 (2025-10-31).

Approximate runtimes for longer simulation chunks are provided throughout the code.
